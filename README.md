# 실습평가서 웹앱

해운대_나눔과행복병원 실습생 평가 시스템

## 배포

Vercel에 이 저장소를 연결하면 자동 배포됩니다.
- Output Directory: `public`

## 구글 스프레드시트 연동

1. `google-apps-script/Evaluation.gs` 코드를 Google Apps Script에 붙여넣고 웹앱으로 배포
2. 웹앱 관리자 탭에서 배포된 Apps Script URL을 등록
3. "스프레드시트에서 불러오기"로 학생 명단 동기화

## 스프레드시트 형식

| A: 분기 | B: 기간 | C: 학교 | D: 학생 | ... | H: 주간평가 |
|---------|---------|---------|---------|-----|------------|
| 2026 춘계 | 3.2~4.24 | 인제대 | 민성연 | ... | O |
